# 2023-05-11-PU Not CPU
I replace the term `CPU` with a different term `PU`.

`CPU` was a term invented in the 1950s when it was believed that an electronic machine, i.e. a *computer*, contained exactly one processing unit and that memory was so expensive that we needed to share it and recycle it (Garbage Collection).

Today, the rules have changed.  We can have many processing units, each with their own local memories.  They are naturally thread-safe.

There is nothing *central* about processing units anymore.  So, I drop the `C` and call these things `PU`s.
