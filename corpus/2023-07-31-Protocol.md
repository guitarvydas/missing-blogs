# 2023-07-31-Protocol
Protocol - A sequence of steps required to synchronize two independent entities. 

- Example: Meet a new person, shake their hand. Protocol. 

- Example: Don't start a meeting until everyone arrives. Protocol. 

- Example: Don't start a meeting until a certain person arrives. Protocol. 

- Example: Start a meeting on time, even if everyone has not yet arrived. Protocol. 

- Example: Send a GET request to a web server. Wait for the response. Protocol. 

- Example: Fork() several process. Wait() for all to finish. Then proceed. Protocol.
