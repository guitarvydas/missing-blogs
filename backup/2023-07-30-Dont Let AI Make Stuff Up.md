# 2023-07-30-Dont Let AI Make Stuff UpDon't let AI make stuff up.

Don't let AI hallucinate.

Keep it simple.

Ask AI to translate your ambiguous instructions into an unambiguous set of instructions and show you the result before taking any action. Then, approve or decline or revise the translated instructions.

Once the instructions appear to be unambiguous, and, correct, ask AI to perform the action.

This means that we have to define what actions a certain machine can take. Then, have AI translate our ambiguous instructions into those, and only those, actions.

We might create more than one kind of machine, each machine able to execute a different set of instructions.
